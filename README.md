# gmail_assistant — Gmail 整理小助手

依寄件者自動在 Gmail 建立巢狀標籤，把收件匣的信件分類到對應資料夾。

## 功能

1. **登入 Google 帳號**：OAuth2 授權，第一次開啟瀏覽器授權，之後自動沿用 `token.json`
2. **掃描信件**：輸入 Gmail 搜尋語法（如 `in:inbox`、`is:unread`），或從下拉選單直接選擇已建立的標籤來查詢該標籤底下的信
3. **寄件者分析**：統計每位寄件者的信件數，列表顯示並依信件數排序；每位寄件者會同時顯示「目前已歸類在哪個標籤」，沒有標籤則留空
4. **套用標籤**：兩種模式
   - **各自分類**（群組標籤欄留空）：自動建立 `寄件者/網域/名稱` 巢狀標籤
   - **群組標籤**（填入自訂名稱）：多個寄件者的信件統一歸入同一個標籤
   - 套用時可勾選「同時加入自動歸類規則」，之後同寄件者的新信會自動套用同樣標籤
5. **批次控制**：套用過程可隨時暫停／繼續／停止
6. **標籤管理**（「標籤管理」分頁）：重新命名、刪除既有標籤；或把信件從某個標籤整批換成另一個標籤
7. **自動歸類**（「自動歸類」分頁）：維護寄件者地址／網域 → 標籤的規則，啟動背景監看後，收件匣有新信就會自動套用標籤（可選擇同時歸檔移出收件匣）

## 標籤結構範例

```
寄件者/
  google.com/
    Google
    Google Workspace
  github.com/
    GitHub
  自訂群組/          ← 群組標籤模式
    電商訂單
```

## 安裝

```bash
pip install -r requirements.txt
```

> 不需要付費。Gmail API 屬於免費配額，個人使用不會產生任何費用。

## 第一次使用設定（只做一次）

1. 前往 [Google Cloud Console](https://console.cloud.google.com)，建立新專案
2. 啟用 **Gmail API**
3. 建立憑證 → OAuth 2.0 用戶端 ID → **桌面應用程式**
4. 下載 `credentials.json`，放到本專案資料夾（已被 `.gitignore` 排除，不會上傳）

## 使用方式

```bash
# GUI 版本（建議）
python gmail_organizer_gui.py

# CLI 版本
python gmail_organizer.py "in:inbox" 200
```

### GUI 操作流程

1. 點「登入 Google 帳號」→ 瀏覽器開啟授權頁，完成後自動關閉
2. **整理信件**分頁：
   1. 在「Gmail 搜尋語法」欄輸入條件（預設 `in:inbox`），設定最多掃描封數
   2. 點「分析寄件者」→ 下方出現寄件者清單（含信件數）
   3. 勾選要整理的寄件者
      - 若要讓多個寄件者共用同一標籤，在「群組標籤」欄填入標籤名稱；留空則依網域自動分類
      - 勾選「同時加入自動歸類規則」可把這次的標籤對應存成規則，供之後新信自動套用
   4. 點「套用標籤」→ 確認後開始執行，進度即時顯示在下方
3. **標籤管理**分頁：點「重新整理標籤列表」載入現有標籤，選取後可「重新命名」或「刪除標籤」；也可在「更換標籤」區把某個標籤底下的信件整批換成另一個標籤
4. **自動歸類**分頁：
   1. 在規則區新增規則（比對寄件者地址或網域 → 套用標籤，可選擇是否同時歸檔）
   2. 設定檢查間隔（分鐘）後按「啟動自動歸類」，背景監看收件匣，符合規則的新信會自動套用標籤
   3. 按「停止」結束監看；下次啟動會接續上次的監看進度，不會重複處理舊信

## 建議工作流程

這個工具的價值在於「一次性整理收件匣裡已經堆積的大量舊信」，這件事 Gmail 網頁介面不好操作（沒辦法依寄件者整批挑選、批次貼標籤）。但「修改標籤」與「讓新信自動分類」這兩件事，Gmail 網頁本身也有原生功能，運作在伺服器端、不需要開著這個程式，通常更省心。建議這樣搭配：

1. **整理舊信（一次性，用本工具）**
   - 「整理信件」分頁掃描 `in:inbox`（或 `in:all` 整理所有信件），依寄件者分組後勾選套用標籤
   - 套用時勾選「同時加入自動歸類規則」，把這次的寄件者 → 標籤對應存下來
2. **修改標籤（兩種都可以）**
   - 本工具的「標籤管理」分頁：重新命名／刪除／整批換標籤，適合大量信件一次處理
   - 或直接在 Gmail 網頁：標籤旁的「⋮」選單 → 編輯／移除，適合單個標籤的小調整
3. **讓新信自動分類（兩種都可以，可只選一種）**
   - **本工具背景監看**（「自動歸類」分頁啟動）：規則存在 `rules.json`，跟整理信件分頁共用；缺點是只有程式開著時才會運作
   - **Gmail 內建篩選器**（設定 → 篩選器和已封鎖的地址 → 建立新篩選器）：伺服器端 24 小時運作，不需要開電腦；缺點是規則要在 Gmail 網頁另外設定一次，不會跟本工具的規則同步
   - 建議：步驟 1 套用標籤後，把同樣的寄件者／網域條件，在 Gmail 篩選器再建一次，之後就可以不用一直開著這個程式

## 常用搜尋語法

| 語法 | 說明 |
|---|---|
| `in:inbox` | 收件匣 |
| `in:all` | 所有信件（含已封存） |
| `is:unread` | 未讀信件 |
| `from:@gmail.com` | 特定網域的來信 |
| `older_than:1y` | 超過一年的舊信 |
| `is:unread in:inbox` | 收件匣未讀 |
